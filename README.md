# Star Tracker: เปลี่ยนความสำเร็จในแต่ละวัน ให้เป็นดาวสะสมในกระปุกส่วนตัว

เคยไหม? ที่รู้สึกว่าแต่ละวันผ่านไปโดยที่ไม่ได้ทำอะไรสำเร็จเลย... เว็บนี้จะช่วยให้คุณได้ ‘เห็น’ ความภูมิใจของตัวเองในทุกๆ วัน บันทึกเรื่องราวดีๆ มอบดาวให้ตัวเอง สะสมแต้มอัปเลเวลเหมือนเล่นเกม พร้อมระบบค้นหาคีย์เวิร์ดและระบบแบ่งหน้าจอเพื่อรองรับบันทึกของคุณได้นับหมื่นดวง!

**ปลอดภัย 100%: เว็บนี้ไม่มีการเก็บรหัสผ่านไว้ในโค้ดหน้าบ้าน แต่ใช้ระบบสมาชิกแท้ๆ ของ Supabase ข้อมูลไดอารี่หรือบันทึกของคุณจะถูกล็อกไว้ในตู้เซฟส่วนตัว ไม่มีใครในโลกแอบดูได้นอกจากคุณ!**

# วิธีสร้าง "Star Tracker" เป็นของตัวเองฟรี! (ใช้เวลาแค่ 10 นาที)

แอปนี้เป็นแบบ Self-hosting แปลว่าคุณจะได้เป็นเจ้าของระบบและฐานข้อมูลทั้งหมดเองฟรีๆ 100% ไม่ต้องแชร์ข้อมูลกับใคร โดยทำตาม 3 ขั้นตอนนี้:

**1. เตรียมหน้าบ้าน (GitHub)**
- กดคัดลอก (Copy) โค้ดไฟล์ index.html จากโปรเจกต์นี้ไป
- สร้าง Repository ใหม่ใน GitHub ตั้งค่าเป็น Public
- อัปโหลดไฟล์ index.html ขึ้นไป แล้วไปที่เมนู Settings -> Pages เปลี่ยน Branch เป็น main แล้วกด Save เพื่อรับ link หน้าเว็บของคุณ

**2. เตรียมตู้เซฟเก็บข้อมูล (Supabase)**
- สมัครเว็บ Supabase.com ฟรี แล้วสร้างโปรเจกต์ใหม่
- ไปที่ Table Editor สร้างตาราง 2 ตารางดังนี้:
- ตารางที่ 1 categories: เพิ่มคอลัมน์ชื่อ name (Type: text) แล้วเปิด RLS ตั้งกฎให้ authenticated อ่านได้ (เอาไว้เก็บรายชื่อหมวดหมู่ที่งอกเพิ่มเองได้)
- ตารางที่ 2 achievements: เพิ่มคอลัมน์ title (text), category (text), stars (int8), success_date (date), และ user_id (uuid ตั้งค่า Default เป็น auth.uid()) จากนั้นเปิด RLS ตั้งกฎให้ตรวจเช็คเฉพาะ auth.uid() = user_id เพื่อล็อกไม่ให้คนอื่นเห็นข้อมูล
- ไปที่เมนู Authentication -> Users เพื่อกดสร้าง Email และ Password ของคุณที่จะใช้ล็อกอินเข้าแอป

**3. เชื่อมต่อระบบเข้าด้วยกัน**
- กลับไปที่ไฟล์ index.html ใน GitHub ของคุณ
มองหาคำว่า SUPABASE_URL และ SUPABASE_KEY (อยู่แถวๆ ด้านล่างของไฟล์)
- นำ URL และ API Key จากหน้าโปรเจกต์ Supabase ของคุณมาวางทับลงไป แล้วกด Save

**เสร็จเรียบร้อย! นำลิงก์จาก GitHub Pages ไปเปิดใช้งานบนมือถือ กด ‘เพิ่มไปยังหน้าจอโฮม’ (Add to Home Screen) คุณก็จะได้แอปสะสมดาวส่วนตัวน่ารักๆ ไว้ใช้งานตลอดไปแล้วล่ะ!**
